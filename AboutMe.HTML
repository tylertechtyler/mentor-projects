They say a picture paints a thousand words, but you know what they say about HTML image maps?

They say click on an object to learn more about it, or at least thats what I say.


No but really image maps are the coolest thing you can do with just HTML

Let me explain what goes into making an image map

  • Pick out an image. (optional: make it interesting)
  • Host your image in a Github repo so you can access it anywhere
  • Set your img's usemap attribute to your image-map's name attribute
  • Set each area of your image-maps coordinates based on what you want to be clickable
  • Use a tool like MS paint, Photoshop, or free online coordinates generator to find the coordinates
  • Set each area's target attribute to "_self" so that it opens in the same frame as your img
  • Don't forget the alt and title attributes.
  • Go to w3schools if you want a decent tutorial
